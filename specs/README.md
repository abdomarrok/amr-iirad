# Amr-Iirad Specifications Directory
This folder contains the complete documentation, architectural blueprints, and legal analysis for the Revenue Order Management System (أمر بالإيراد).

### 📖 Reading Guide (Chronological Order)

For a full understanding of the project, please read these files in the following order:

1.  **[instruction_08_2023_analysis.md](./instruction_08_2023_analysis.md)**  
    *The Legal Foundation.* Full extraction and field-by-field analysis of **Instruction n° 08 (09 April 2023)**. Read this first to understand the mandatory regulatory requirements.

2.  **[PRD.md](./PRD.md)**  
    *The Product Vision.* High-level requirements, target users, and core functional scope based on the legal framework.

3.  **[amr_iirad_miniapp_plan.md](./amr_iirad_miniapp_plan.md)**  
    *The Blueprint.* Detailed project plan including database schema, business rules, and technical architecture. This is the primary reference for implementation logic.

4.  **[technical_manifest.md](./technical_manifest.md)**  
    *The Engineering Fingerprint.* **CRITICAL FOR DEVELOPERS.** Summarizes the current architectural state, modular patterns, design system principles (GstockDz look), and non-negotiable coding standards.

5.  **[advancement.md](./advancement.md)**  
    *The Progress Log.* A chronological record of all development milestones, completed phases, and resolved technical challenges. Check this to see what has been built and what is left.
